Detailed Description

This Model facilitates the creation of the SellsTogetherReport.

This model aggregates the amount sold of every pair of Menu items for each day. This makes aggregating the SellsTogetherReport easy, at the expense of slightly slowing down the checkout time for each order and taking marginally more space on the DB.

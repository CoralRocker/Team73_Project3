Detailed Description

OrderItem Model Class.

An OrderItem is a single menu item which belongs to an order. The OrderItem may have as many customizations added to it as the customer wants, and as many as desired can be ordered.

The model is related to the Menu Model, the Order Model, and the Customizations Model. The model uses an intermediate ItemCustomization model to handle the customizations for each item.
